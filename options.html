Hiring Cafe Watcher

Configure the keyword to monitor, email delivery settings, and the run interval.

Used for https://hiring.cafe/api/search-jobs?search=<keyword> requests.
Credentials are stored locally in Chrome storage and used only by this extension to send mail.
Mails are sent through https://smtpjs.com/v3/smtpjs.aspx using the SMTP settings above.
Minimum interval is 15 minutes. Defaults to 120 minutes (2 hours).
